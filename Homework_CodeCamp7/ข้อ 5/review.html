5.1 position คืออะไร และ อะไรบ้าง

      ตอบ ใช้กำหนดตำแหน่งของ element ในแบบต่างๆ มีดังนี้

Position Property Description
Static กำหนด element ให้คงที่ ไม่สามารถเคลื่อนที่ไปไหนได้
Relative กำหนดตำแหน่ง element ได้ว่าจะให้มีระยะห่างจาก top,left,right,bottom เท่าไร
Fixed กำหนดำแหน่ง element ว่ามีระยะห่างจากจุดต่างๆ คล้ายๆ Relative
แต่กันตรงที่ว่า ไม่ว่าจะ scroll เมาส์ขึ้นหรือลง element จะถูก fixed ไม่เลื่อนขึ้นหรือลงตาม
Absolute กำหนดตำแหน่ง element ว่าระยะห่างจาก top,left,right,bottom เท่าไร
โดยอิงจาก element ที่อยู่ใกล้กัน
Sticky ทำหน้าคล้าย Fixed และ Relative แต่ต่างจาก Fixed ตรงที่ว่า
element ไม่ได้ถูก fixed ตำแหน่งไว้ เพียงแต่ถ้า scroll เมาส์ ขึ้นหรือลง จนสุดขอบ element ที่ถูก sticky จะเลื่อนตามขอบจอขึ้น/ลงตามมาด้วย
