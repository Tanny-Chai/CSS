2.1 property และ value คืออะไรแตกต่างกันอย่างไร

Type Description
Property คือ ส่วนของคุณสมบัติที่เราต้องการให้แสดง เช่น สีตัวอักษร ชนิดตัวอักษร ขนาดตัวอักษร
Value คือ ค่าที่เราต้องการให้แสดง เช่น ตัวอักษรสีแดง ตัวอักษรแบบ Tahoma ตัวอักษรขนาด 12pt

2.3 value unit มีกี่ประเภทอะไรบ้างแต่ละ ประเภทคืออะไร

      ตอบ มี 2 ประเภทดังนี้

Type Descripton
Absolute Lengths cm = centimeter
mm = millimeter
in = inches
px = pixel
pt = point
pc = picas
Relative Lengths em = กำหนดระยะห่างระหว่างบรรทัด เช่น
2em จาก 16 px หมายถึง เว้นระยะห่างระหว่างบรรทัด 16x2 = 32px
ex = เพิ่มความสูงตามแนว x-height
ch = เพิ่ม/ลด ขนาดตัวอักษร เทียบกับ 0
rem = เพิ่ม/ลด ขนาดตัวอักษร โดยเทียบกับการถอดรากของสเกลเดิม
เช่น 2rem จาก 16px หมายถึง ลดขนาดตัวอักษรจาก 16px ให้มีขนาดเป็นรากที่สอง ของ 16 คือ 4px
vw = เพิ่ม/ลด ขนาดตัวอักษรเทียบกับ % ความกว้างของ viewport นั้นๆ
ถ้ามีการเพิ่มหรือลด ขนาด viewport ตัวอักษรก็จะพูดเพิ่มหรือลดตามไปด้วย
เช่น 20vw หมายถึง ตัวอักษรนี้มีขนาด 20% เทียบกับความกว้างของ viewport ขณะแสดง
vh = เพิ่ม/ลด ขนาดตัวอักษรเทียบกับ % ความสูงของ viewport นั้นๆ
ทำหน้าที่คล้ายๆ vw ต่างกันแค่จุดอ้างอิง
เช่น 20vh หมายถึง ตัวอักษรนี้มีขนาด 20% เทียบกับความสูงของ viewport ขณะแสดง
vmin = เพิ่ม/ลด ขนาดตัวอักษรเทียบกับ % ความกว้างยาวของ viewport ที่มีขนาดเล็กที่สุด
หมายความว่า ถ้าขยาย viewport ให้ใหญ่กว่ามาตรฐาน ตัวอักษรจะไม่ขยายใหญ่ตาม
แตุ่ถ้าย่อให้เล็กกว่าขนาดมาตรฐาน ตัวอักษรจะย่อตาม
vmax = เพิ่ม/ลด ขนาดตัวอักษรเทียบกับ % ความกว้างยาวของ viewport ที่มีขนาดใหญ่ที่สุด
หมายความว่า ถ้าย่อ viewport ให้เล็กกว่ามาตรฐาน ตัวอักษรจะไม่ย่อตาม ถ้าขยาย viewport
ตั้งแต่มาตรฐานขึ้นไปเรื่อยๆ ตัวอักษรจะถูกขยายตามไปเรื่อยๆ
% = เพิ่มขนาดตัวอักษรเทียบ % จากสเกลเดิม
หมายความว่า 100% หมายถึง ตัวอักษรขนาดเท่าเดิม
>100% หมายถึง ตัวอักษรถูกขยาย
<100% หมายถึง ตัวอักษรถูกย่อ
