1.1 CSS syntax ประกอบด้วย

Syntax Description
1.Selector กำหนด HTML element ที่ต้องการปรับ style
2.Declaration อยู่ถัดจาก Selector ประกอบด้วย Property name และ Value
คั่นกันด้วยเครื่องหมาย :
แต่ละ Declaration จะถูกคั่นด้วยเครื่องหมาย ;
และ Declaration block จะถูกล้อมรอบด้วยวงเล็บปีกกา
2.1 Property รายละเอียดการกำหนด style เช่น สีตัวอักษร สีพื้น ขอบ ความกว้าง ความยาว เป็นต้น
2.2 Value ส่วนขยายที่ระบุุว่ากำหนด Property เป็นอย่างไร เช่น ให้สีตัวอักษรเป็นสีแดง สีพื้นเป็นสีเหลือง เป็นต้น

Embed

Style

Sheet		 เขียน Style โดยตั้ง Element style ภายใน Element <head>
และเขียน Declaration ด้านหลัง Element ซึ่งเราสามารถกำหนดเป็น
class หรือ id ก็ได้ ซึ่งจะแสดงตัวอย่างไว้ในช่อง Example
และนำ style นั้นมาใส่ใน Element ใน <body> อีกที
เหมาะสำหรับ การตั้ง style เดียวกัน ในหลายๆ Element 		กำหนด style ที่ได้ผลลัพธ์เดียวกัน ในแต่ละแบบ
แบบที่ 1 ไม่กำหนด class หรือ id ใดๆ

<head>
   <style>
       p {
          font-size:30px;
          color:#FADBD8;
       }
   </style>
</head>

<body>
       <p> สวัสดีค่ะ</p>
</body>

การกำหนดแบบนี้ หมายความว่า ทุก element <p>
จะมี style เช่นนี้ทั้งหมด

แบบที่ 2 กำหนด id ใน style

<head>
   <style>
       #abc {
          font-size:30px;
          color:#FADBD8;
       }
   </style>
</head>

<body>
       <p id="abc"> สวัสดีค่ะ</p>
</body>

การกำหนด id จะทำให้ style ปรากฏแต่เฉพาะ element ที่กำหนด id เท่านั้น

แบบที่ 3 กำหนด class ใน style

<head>
   <style>
       .abc {
          font-size:30px;
          color:#FADBD8;
       }
   </style>
</head>

<body>
       <p class="abc"> สวัสดีค่ะ</p>
</body>

การกำหนด class จะทำให้ style ปรากฏแต่เฉพาะ element ที่กำหนด class เท่านั้น
