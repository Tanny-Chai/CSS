1.1 CSS syntax ประกอบด้วย

Syntax Description
1.Selector กำหนด HTML element ที่ต้องการปรับ style
2.Declaration อยู่ถัดจาก Selector ประกอบด้วย Property name และ Value
คั่นกันด้วยเครื่องหมาย :
แต่ละ Declaration จะถูกคั่นด้วยเครื่องหมาย ;
และ Declaration block จะถูกล้อมรอบด้วยวงเล็บปีกกา
2.1 Property รายละเอียดการกำหนด style เช่น สีตัวอักษร สีพื้น ขอบ ความกว้าง ความยาว เป็นต้น
2.2 Value ส่วนขยายที่ระบุุว่ากำหนด Property เป็นอย่างไร เช่น ให้สีตัวอักษรเป็นสีแดง สีพื้นเป็นสีเหลือง เป็นต้น
