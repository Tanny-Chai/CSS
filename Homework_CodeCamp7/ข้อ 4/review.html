4.1 value ใน display property มีอะไรบ้าง

Display Property Description Code Example Result
display:inline มักใช้ใน <li> โดยปกติการเขียน <li> ภายใน <ul> หรือ <al> หลาย <li>
จะเรียงจากบนลงล่าง แต่ถ้าใช้ display:inline <li> จะเรียงจากซ้ายไปขวาแทน 		<head>
<style>
     li {
         display:inline;
}
</style>
</head>

<body>

         <li>CSS</li>
         <li>JavaScript</li>
     </ul>
</body>

  • JavaScript
display:block ทำให้ element ที่เรียงต่อเนื่องในบรรทัดเดียวกันกลายเป็น block
เช่น <span> ปกติ ถ้าอยู่คนละ <span> จะอยู่ในบรรทัดเดียวกัน แต่ถ้าใช้ display:block จะอยู่คนละบรรทัดกัน 		<head>
<style>
     span {
         display:block;
     }
</style>
</head>
