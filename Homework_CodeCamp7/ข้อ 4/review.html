4.2 grid และ flex แตกต่างกันอย่างไร

      ตอบ flex และ grid มีหน้าที่หลักๆคล้ายกัน คือ การวาง layout บนหน้าเว็บ แต่จะมีข้อแตกต่างกันบางประการดังนี้

Type Difference
Flex สามารถยืดขยาย cell ได้ และปรับ layout ได้ทีละ 1 มิติ คือ ปรับ
row หรือ column ให้ยืดหรือหด อย่างใดอย่างหนึ่งเท่านั้น
Grid ไม่สามารถยืดขยาย cell ได้ ทำได้เพียงผสาน cell เพื่อให้มีขนาดใหญ่กว่า cell อื่น
เท่านั้น และสามารถปรับ layout ได้พร้อมกันทั้ง 2 มิติ คือ ปรับผสาน cell
ทั้ง row และ column ได้ในคราวเดียวกัน หรือ อย่างใดอย่างหนึ่งก็ได้

4.3 justify and align แตกต่างกันอย่างไร

Type Difference
Justify ใช้จัดแนวตัวอักษรหรือวัตถุตามแนวนอน เช่น ปรับให้ตัวอักษรชิดขอบซ้าย ชิดขอบขวา หรือ อยู่กึ่งกลาง ระหว่างขอบซ้ายขอบขวา
Align ใช้จัดแนวตัวอักษรหรือวัตถุตามแนวตั้ง เช่น ปรับให้ตัวอักษรชิดขอบบน ชิดขอบล่าง หรือ อยู่กึ่งกลาง ระหว่างขอบบนขอบล่าง
